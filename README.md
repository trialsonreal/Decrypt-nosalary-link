SalaryDecrypt

<h3 align="center">
Just for decrypting MyTV's nosalary
</h3>

## How to install
1. Install NodeJS
2. In folder with `main.js`, run `npm i`

## How to uses
1. Get API from OTT source (example MyTV), modify whatever you want
2. Get `nosalary` string
3. Run the code, put `nosalary` string
4. Profit
